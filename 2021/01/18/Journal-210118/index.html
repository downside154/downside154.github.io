    Computer Science and Engineering

    What is the fundamental difference between computation and calculation?

    • Calculation: Implies an array of arithmetic process.
    • Computation: Implies the implementation of rules in a systematic way.

    Computer Science and Engineering (CSE) - The science that deals with computer software and programs

    Academic programs vary between colleges. Courses usually include introduction to programming, introduction to algorithms and data structures, computer architecture, operating systems, computer networks, parallel computing, embedded systems, algorithms design, circuit analysis and electronics, digital logic and processor design, computer graphics, scientific computing, software engineering, database systems, digital signal processing, virtualization, computer simulations and games programming.

    Theoretical CSE

    CSE programs also include core subjects of theoretical computer science such as theory of computation, numerical methods, machine learning, programming theory and paradigms.
    Modern academic programs also cover emerging computing fields like image processing, data science, robotics, bio-inspired computing, computational biology, autonomic computing and artificial intelligence.
    [exerpt from wikipedia]
    More info: CSE

    Process of Computational Thinking

    1. Problem Formulation (Abstraction)
    2. Solution Expression (Automation)
    3. Solution Execution & Evalutation (Analysis)

    Characteristics of Computational Thinking

    1. Decomposition (문제 분해)
    2. Pattern Recognition/Data Representation (패턴인지/데이터 표현)
    3. Generalization/Abstraction (일반화/추상화)
    4. Algorithms (알고리즘)

    Python Enhance Proposal

    An agreed upon format to code in python.

    Layout

    1. Indentation(Tabs) = 4 spaces
    2. Each line is 79 character long (up to 120 is acceptable)
    3. Defining a Class, or function(최상위 함수), is separated with two lines
    4. Methods inside a class are separed with one line

    Variables

    1. _variable: is a given name to be used internally
    2. print_ : to avoid clashing with Python

    Naming Convention

    1. Class: CamelCase
    2. Function, Variable, Method: snake_case

    Not used in Python

    1. chHungarianNotation
    2. javaScriptStyleCamelCase
